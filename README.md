## Lab Questions

### What is difference between edge and level sensitive circuits?
The difference between edge and level sensitive circuits is that edge circuits only react when it changes from 0 to 1 and level circuits react whenever it is at 1.
### Why is it important to declare initial state?
It is important to declare initial state so that the program knows where to start from and does not try to use a value that does not exist at the start of the program.
### What do edge sensitive circuits let us build?
They let us build most modern computer circuits as they stop infinite loops in the circuit while still allowing us to change things on a repeated interval.
